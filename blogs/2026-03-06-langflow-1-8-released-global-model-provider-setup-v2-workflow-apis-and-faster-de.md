Langflow 1.8 reduces credential sprawl with the global model provider setup, improves programmatic integration with V2 workflow endpoints (Phase 1 API redesign, beta), and speeds up troubleshooting with a new traces feature, Inspection Panel, and a more responsive chat experience. Plus knowledge bases, Mustache templating, new components, and modular dependency installation.
