Langflow 1.9 introduces new capabilities for building, operating, and integrating AI workflows, with updates focused on in-product AI assistance, flow deployment tooling, and MCP-based interoperability. This release adds native support for AI-assisted component generation, standardized tooling for managing flows outside the visual builder, and new interfaces that allow IDEs and external agents to create and execute Langflow flows programmatically.
